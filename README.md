<h1>Parallel Send </h1>
<h2>A Python Program that allows to write multiple user defined messages that can be send as many time as the user wants to every writable text box!</h2>

<b>Usage:</b>
<ul>
  <li>Send Unlimited/Limited Messages on Facebook,Messenger,Telegram or Any other Messenger.</li>
  <li>Post Automatic Comments</li>
</ul>
